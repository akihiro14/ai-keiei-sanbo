# 事業検証レポート：MeetMemo
**会議録音をAIが自動で議事録化し、タスクを担当者に通知するSaaS**

作成日：2026-06-03

---

## 1. 課題仮説の検証

### この課題は実際に多くの人が抱えているか

**結論：Yes。定量的に裏付けられた、広範・深刻な課題。**

#### 主要データ

| 指標 | 数値 | 出典 |
|------|------|------|
| 議事録作成にかかる時間（ビジネスパーソン平均） | **年間 319.6時間（週6.13時間）** | キヤノンMJ調査（2022年） |
| 20代が費やす時間 | 週平均 **8.46時間** | 同上 |
| 議事録作成に「負担を感じる」割合 | **67%** | 同上 |
| AIサポートツールの使用を「希望する」割合 | **70%** | 同上 |
| 実際にDX・AI化が進んでいる現場の割合 | **わずか1.4%** | 同上 |
| AI議事録導入後の作成時間削減率 | 平均 **82%削減** | 各ツール導入事例 |

#### 課題の構造

```
録音 → 文字起こし（手動） → 要約・整理（手動） → 担当者へ共有（手動） → タスク追跡（手動）
          ↑                        ↑                        ↑
     時間がかかる             抜け漏れが発生           通知されず忘れられる
```

**「AIを使いたいが使えていない」というギャップ（70% vs 1.4%）が最大の市場機会。**
特に「タスクが会議で決まっても誰も通知されない・追跡されない」という課題は、既存ツールでも未解決のペインである。

### TAM・SAM・SOM の概算

```
TAM（獲得可能な全体市場）
  国内の「会議が発生するビジネスパーソン」＝約3,000万人
  × 月額3,000円（推定ARPU）= 約1兆800億円/年

SAM（狙えるセグメント市場）
  中小〜中堅企業（従業員10〜1,000名）のホワイトカラー
  ≒ 全体の10〜15% → 約1,000〜1,600億円/年

SOM（現実的な獲得目標）
  参入後3〜5年でシェア1〜3% → 約10〜48億円/年
```

### 市場の成長トレンド

- **テレワーク・ハイブリッドワークの定着**：録音・文字起こしニーズが恒常化
- **生成AI（LLM）の普及**：文字起こし精度・要約品質が飛躍的に向上
- **会話インテリジェンス市場**（次フェーズ）：278億USD → 520億USDへの成長が予測され、AI議事録はその入口に位置する
- **DX投資の増加**：企業のAI利活用が加速（総務省 令和7年版 情報通信白書）

### 競合①：Notta

**サービス概要**
日本語を含む58言語に対応した音声文字起こし・AI議事録ツール。日本語認識精度98.86%を誇り、日本市場でシェアが高い。8百万人以上のユーザーを持つ。

**料金プラン**

| プラン | 月額 | 主な制限・機能 |
|--------|------|---------------|
| Free | 無料 | 120分/月、1録音3分まで |
| Pro | $13.99（年払い$8.17） | 1,800分/月、AI要約・タスク抽出 |
| Business | $27.99/シート（年払い$16.67） | 無制限、CRM連携、チーム管理 |
| Enterprise | 要問合せ | SAML SSO、監査ログ、専任サポート |

**強み**
- 日本語精度が業界最高水準（98.86%）
- Zoom/Teams/Google Meet/Webex すべてに対応
- 多言語（58言語）対応で国際チームにも使える
- AI要約・アクションアイテム抽出機能あり

**弱み**
- タスクの「担当者通知」機能が弱く、抽出止まり
- CRM連携はBusinessプラン以上（個人・小規模チームには高コスト）
- 無料プランの3分制限は実用に耐えない

### 競合②：Fireflies.ai

**サービス概要**
60以上の言語対応。独自AIアシスタント「Fred」を搭載し、CRM・Slack・Notion・Asana等との統合が充実。英語圏のセールスチームに強い。

**料金プラン**

| プラン | 月額 | 主な制限・機能 |
|--------|------|---------------|
| Free | 無料 | 800分/月、AI要約3件/月 |
| Pro | $10/ユーザー | 無制限文字起こし、AIサマリー |
| Business | $19/ユーザー | 動画録画、会話インテリジェンス |
| Enterprise | $39/ユーザー | カスタムSSO、専任サポート |

**強み**
- 豊富なインテグレーション（Zapier経由で数百ツールと連携）
- CRM（Salesforce/HubSpot）との連携が強力
- Pro $10/月は業界最安水準のコスパ
- 60言語対応で多国籍チームに対応

**弱み**
- **日本語対応が弱い**（英語最適化）
- 動画録画はBusinessプラン以上（$19）
- タスク通知は「抽出」まで。担当者への自動プッシュ通知はない
- 無料プランのAI要約3件/月は事実上使えない

## 4. 差別化ポイントの整理

### 競合との比較で自社が勝てる軸

---

#### 差別化軸①：「議事録作成」から「タスク完了」までの一気通貫

**競合の状況**
既存ツールはすべて「議事録を作る」「タスクを抽出する」で止まっている。担当者へのプッシュ通知・進捗追跡・リマインダーは未実装か弱機能。

**MeetMemoの提供価値**
```
録音 → 自動文字起こし → AI要約 → タスク抽出 → 担当者に Slack/メール通知
                                                         ↓
                                              タスク完了報告 → 自動アーカイブ
```

会議で決まったことが「自動で担当者に届き、完了まで追跡される」ことで、フォローアップ工数をゼロにする。

---

#### 差別化軸②：日本のビジネス慣行に完全最適化

**競合の状況**
グローバルツール（Fireflies・tl;dv）は英語最適化。Nottaは日本語精度は高いが、UIや通知設計が日本企業の慣行に合っていない。

**MeetMemoの提供価値**
- 日本語議事録フォーマット（件名・出席者・決定事項・TODO）をデフォルト出力
- Slack / Chatwork / LINE WORKSなど国内チャットツールとのネイティブ連携
- 「担当者＋期日」を明示した日本語タスク通知テンプレート
- 敬語・ビジネス語の文体補正機能

---

#### 差別化軸③：導入摩擦ゼロ設計（ノーコード×既存ツール接続）

**競合の状況**
導入率が1.4%にとどまる最大の理由は「使いこなせない」「IT部門の承認が必要」という導入障壁。

**MeetMemoの提供価値**
- 専用アプリ不要：Google Meet / Zoom / Teams のボット招待だけで即日利用開始
- IT部門の関与なしに現場部門が自己完結で導入可能
- 月額3,000円以下の個人・小チーム向け低単価エントリープランで試用ハードルを排除
- 無料トライアル14日間（データ削除なし）で導入判断を後押し
