# 事業計画書（草案）

## MeetMemo
**〜会議の録音をAIが議事録化し、タスクを担当者に届けるSaaS〜**

作成日：2026-06-04
ステータス：草案（内部検討用）

---

## 1. エグゼクティブサマリー

### 事業概要

MeetMemoは、**中小企業の「会議後の無駄」を丸ごと解決するAI議事録SaaS**である。
会議の録音をAIが自動で文字起こし・要約し、決定したタスクを担当者のSlack・Chatwork・メールへ自動通知する。「議事録を書く」で終わる既存ツールと異なり、**タスクが完了するまでをワンストップでカバー**する点が最大の差別化軸である。

### ターゲット

従業員50名以下の中小企業（日本国内）。専任IT担当者がおらず、議事録作成を若手社員が手作業で担っている現場。

### ビジネスモデル

月額課金（サブスクリプション）。¥3,000/ユーザー/月。

### 数値目標（3年）

| 指標 | 1年目末 | 2年目末 | 3年目末 |
|------|--------|--------|--------|
| 導入社数 | 150社 | 500社 | 1,200社 |
| ARR | 1,620万円 | 5,400万円 | 1.3億円 |
| 損益 | ▲1,000万円 | +1,700万円 | +6,500万円 |

### なぜ今か

- AIによる議事録自動化は技術的成熟期に入ったが、**導入率はまだ1.4%**
- 国内AI議事録市場は年率42%で急成長中（2026年推計380億円）
- IT導入補助金2025の拡充で中小企業の導入障壁が低下している

## 2. 課題と解決策

### 2-1. 課題：会議後の「見えないコスト」

日本のビジネスパーソンは議事録作成に**年間平均319.6時間**を費やしており、67%が業務負担として認識している（キヤノンマーケティングジャパン調査）。しかし課題はそれだけではない。

```
会議が終わった後に起きていること
─────────────────────────────────────────────
❌ 誰かが手で議事録を書く（平均1.5〜2時間/回）
❌ 決まったタスクが担当者に伝わらない・忘れられる
❌ フォローアップが属人化し、抜け漏れが繰り返される
❌ 若手社員が議事録番として会議に集中できない
─────────────────────────────────────────────
```

特に従業員50名以下の中小企業では、これらの作業を担う専任スタッフがおらず、問題が放置されやすい。

### 2-2. 既存ツールの限界

市場にはNotta・LINE WORKS AiNoteなど多数のAI議事録ツールが存在するが、**すべてが「議事録を作る」で止まっている**。タスクの抽出はできても、担当者への自動通知・完了追跡まで担うツールは現時点で存在しない。

### 2-3. MeetMemoの解決策

```
【会議前】
  Zoom / Google Meet / Teams にMeetMemoボットを招待（30秒で完了）

【会議中】
  自動録音 → リアルタイム文字起こし

【会議後・自動】
  ┌─────────────────────────────────────────┐
  │  AI要約（3行サマリー）                    │
  │  決定事項の構造化出力                     │
  │  タスク自動抽出（誰が・何を・いつまでに） │
  │         ↓                                │
  │  担当者のSlack / Chatwork / メールに通知  │
  │         ↓                                │
  │  完了報告 → 自動アーカイブ               │
  └─────────────────────────────────────────┘
```

**提供価値の本質**：議事録の「作成」ではなく、タスクの「完了」まで責任を持つ。

## 3. 市場規模（TAM・SAM・SOM）

### 3-1. 前提データ

| データ | 数値 | 出典 |
|--------|------|------|
| 日本の企業数（全体） | 約368万社 | 経済センサス（2021年） |
| うち従業員50名以下の比率 | 約**76%**（≒280万社） | 中小企業庁 |
| 中小企業のSaaS利用率 | 2025年 **48%**（前年+10pt） | MM総研 |
| AI議事録ツール導入率 | **1.4%** | キヤノンMJ調査 |
| 国内AI議事録市場（2026年） | **約380億円**（前年比+42%） | 各調査機関推計 |
| グローバル市場（2025年） | **約31〜35億USD** | Grand View Research |
| グローバル市場CAGR | **約24〜26%** | Market.us / Precedence |

### 3-2. TAM・SAM・SOM

```
TAM（Total Addressable Market）：獲得可能な全体市場
─────────────────────────────────────────────────────
  国内の「会議が発生する中小企業（50名以下）」
  = 280万社 × SaaS利用率48% × 平均3ユーザー × 月額3,000円 × 12ヶ月
  ≒ 約1,450億円/年

  ※ グローバル市場は31〜35億USDで高成長中
```

```
SAM（Serviceable Addressable Market）：現実的に届く市場
─────────────────────────────────────────────────────
  Zoom / Google Meet / Teams 利用中 かつ
  Slack / Chatwork 通知環境が整っている中小企業（50名以下）
  = TAMの約25% ≒ 約360億円/年
```

```
SOM（Serviceable Obtainable Market）：3年以内の現実的獲得目標
─────────────────────────────────────────────────────
  シェア0.3〜0.5% ≒ 約1〜2億円/年（ARRベース）
  ＝ 1,200社 × 平均月額9,000円（3ユーザー） × 12ヶ月 ≒ 1.3億円
```

### 3-3. 市場成長トレンド

- 国内AI議事録市場：**年率42%**で拡大中（2026年）
- グローバル：2025年の31〜35億USDが2035年には63〜343億USDへ（CAGR 18〜26%）
- 成長ドライバー：テレワーク定着・生成AIの精度向上・人手不足問題の深刻化

### 5-2. 競合各社の詳細

**Notta**
- 日本語精度98.86%・58言語対応。8百万ユーザー。
- Pro $13.99/月（年払い）から利用可能。
- **弱点：タスク担当者通知なし。CRMはBusinessプラン以上（$27.99/シート）**

**Otter.ai**
- 英語特化。アクションアイテムの割り当て機能あり。
- Pro $16.99/月（月払い）
- **弱点：日本語対応が弱い。英語圏向け設計**

**LINE WORKS AiNote**
- LINE WORKSと統合。文字正解率90.8%。業種特化モデルあり。
- チームプラン年払い月額19,800円（月払い22,000円）、個人向けソロプラン月1,440円〜
- **弱点：LINE WORKSを使っていない企業には不向き。タスク通知機能は未実装。チームプランが高価（月2万円）**
